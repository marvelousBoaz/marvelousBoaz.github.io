静态展示页面的服务

通常使用html、markdown文件
需要条件：username.github.io

静态站点建站系统

-octpress (ruby)
-pelican(python)
-hexo (node.js)

安装过程：

-node.js npm(包管理器，相当于python中的pip)
-sudo npm install -g hexo-cli

使用

-初始化：hexo init
-创建：hexo n “标题“
-生成：hexo g
-预览：hexo s

发布到git

-使用hexo一键发布: hexo d
-手动push方式: 将publish目录，git push origin master
